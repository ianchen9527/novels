# 待決定

尚未拍板、但會影響很多後續的決定。大致按重要性排。

## 1. 失控的性質（決定全書的顏色）

- **甲 技術性失控**：提純本身中性，失控是修行者貪快、根基不穩、或運氣不好。世界沒有惡意，悲劇是人性的，偏成長冒險（獵命師那種熱血奇想）。
- **乙 設計性失控**：失控是「回收」機制的一部分，「越修越渴望本源」是被設計出來的誘餌——整個修行體系是一座用人肉提純力量的養殖場。世界有惡意，悲劇是結構性的，偏權謀揭密，黑色調重（慶餘年那種笑著看透的涼薄）。
- 附帶未定：失控的**具體形式**（爆體／變怪物／其他）。

## 2. 修行的門檻：一個人憑什麼能修行（決定社會結構）

- **天賦可測**：能檢測誰有資格 → 有「選拔」，也有一大批「差一點就過、終究沒過」的普通人。這批落選者是普通人／修行者張力的金礦。
- **被本源／組織選中**：資格握在教團手裡，能不能修行不只看天分，還看關係、看政治 → 走後門、冤案，權謀味最重。
- **罕見體質／機緣**：某種天生體質，或某個觸發事件（瀕死、際遇）才踏得上 → 更隨機神祕，主角的入門本身可以是個謎。

## 3. 提純「利息」的本質（決定主題）

人的一生到底加工了什麼，讓還回去的力量比借來的多？

- **經歷**——力量沾了情感、記憶、意志，「活過的痕跡」本身是養分。（貓膩味）
- **痛苦**——加工的是苦難，修行者受的每一份罪都是利息。（黑，獻祭味）
- **創造**——本源無意識、不會創造，修行者用力量做出本源做不到的事，那份「新意」是利息。（哲學味最重）

## 4. 主角借哪個本源入門

挑一個能讓世界結構自然展開的本源，未必要強或酷。有時最普通的本源最好帶讀者。
