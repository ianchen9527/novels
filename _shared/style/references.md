# 參照作品與品味基準

跨書共用。寫任何一本都適用。

## 作者自述的共通點

1. 人物刻畫細膩，沒有免洗角色
2. 場景有電影分鏡感
3. 設定合理，不戰力膨脹
4. 角色間張力強，有衝突與代入感

## 萃取出的設計原則

### 制度先於角色

六本全部把主角放進一個組織裡：監察院、書院、打更人衙門、武館門派。衝突因此不是善惡，是**立場**——上司、同僚、規矩、政治風險。這是角色張力的真正來源。

免洗角色之所以免洗，是因為他在制度裡沒有位置，只是主角路上的障礙物。

**設計順序：先制度，再角色在制度裡的位置。**

### 力量必須有價格

「不戰力膨脹」是症狀，機制是**每強一分都要付出讀者看得見的東西**。三位作者收不同的貨幣：

- 喬靖夫收物理（人體極限、距離、重心、體力）
- 貓膩收天賦與痛苦（寧缺天賦差，每一步都是苦功換的）
- 賣報小郎君收政治風險（查得越深，得罪的人越大）

膨脹難看不是因為數字大，是因為**變強變成免費的**。

### 沒有免洗角色 = 每個有名字的角色都有跟主角無關的目標

可操作的檢查：**這個角色在主角不在場時想要什麼？** 答不出來就是免洗。

### 兩種筆法，兩種段落

貓膩的恬淡日常與喬靖夫的高密度動作是不同的呼吸，不衝突，是分工：

- **積累段落用貓膩式**：慢、生活感、對話養人物、埋線
- **引爆段落用喬靖夫式**：鏡位在動、距離與重心、時間拉長、物理後果

這正好對應「恬淡積累後引爆」的節奏偏好。

注意：恬淡不等於沒事發生。積累段落必須一直在給讀者具體、有質感的細節，否則就是無聊。這是這個結構唯一的風險。

### 電影分鏡感 = 攝影機有鏡位

不是「描寫得很細」，是鏡位會動：遠景交代空間 → 中景給動作 → 特寫給細節或情緒。動作場面的時間可以被拉長。喬靖夫的招牌是打鬥中的**距離感與重心轉移**。

場景描寫的原則仍然是密度而非篇幅：挑一兩個有感官特異性的細節當支點，不要全景描寫。

### 章節結構：案件驅動

參照大奉打更人——一個案子 = 一個小的積累→引爆循環，同時往前推一格主線。玄幻的世界觀在案子後面慢慢展開，推理與懸疑是讓人翻頁的引擎。

連載節奏與「恬淡後引爆」的偏好在這個結構下不打架。
